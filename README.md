# Plankton VPN MVP

Bilingual EN/RU MVP structure for a branded VPN product for the $PLANKTON community.

## Stack
- Next.js website
- TonConnect wallet connection
- TON jetton balance check placeholder
- Telegram bot skeleton
- PostgreSQL + Prisma schema
- WireGuard / wg-easy integration placeholder

## Quick start
```bash
npm install
cp .env.example .env
npm run dev
```

## Pages
- `/en` — English landing
- `/ru` — Russian landing
- `/en/dashboard` — user dashboard concept
- `/ru/dashboard` — user dashboard concept
- `/en/admin` — admin concept
- `/ru/admin` — admin concept

## Important
This is a product scaffold, not a production VPN engine. Use WireGuard/wg-easy or another trusted VPN engine.
